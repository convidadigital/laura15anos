PROJETO CHAVE MÁGICA - GITHUB PAGES

Arquivos incluídos:
- index.html
- style.css
- script.js
- porta.png
- chave.png
- frame-correto.png

Falta você adicionar seu vídeo com este nome exato:
- chave-girando.mp4

Como publicar:
1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. Adicione seu vídeo MP4 renomeado como chave-girando.mp4.
4. Vá em Settings > Pages.
5. Em Build and deployment, escolha Deploy from a branch.
6. Escolha main / root e salve.
